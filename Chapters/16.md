# Modal Presentation

### State 
```
@State var showUpdate = false
```

### Button 
```
Button(action: { self.showUpdate.toggle() }) {
    Image(systemName: "bell")
        .renderingMode(.original)
        .font(.system(size: 16, weight: .medium))
        .frame(width: 36, height: 36)
}
```

### Presentation Modal
- To call a modal, you need to use a `.sheet` modifier. Whatâ€™s amazing is that this will work for iPad as well and SwiftUI takes care of adapting the modal frame. However, there are built-in gestures so make sure that your content doesn't rely on custom gestures.

```
@State var showUpdate = false

// Button
.sheet(isPresented: $showUpdate) {
    ContentView()
}
```
